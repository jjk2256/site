Unpredictable 2024

What began as a playful homage to Rube Goldberg machines quickly revealed how messy physical computation really is. Every tiny variable—airflow, friction, spacing, sensor delay—twisted a clean input → process → output into a different outcome each run. In the end, that unpredictability became the project's real expression, showing me how misfires give a system its character.

Unpredictable is a kinetic chain reaction machine created as an homage to the tradition of the Rube Goldberg machine, where simple actions trigger overengineered sequences of events. The project examines how a seemingly straightforward input → process → output system becomes unstable once translated into physical motion.

The sequence begins with a trigger (input): a lever strike that sends dominoes down a track. Their motion activates a sensor connected to an Arduino (process), which powers a fan and initiates additional digital cues. These cues pull a balloon toward a needle field, forming the intended output—popping the balloon.

However, the system continually resists expectation. Small variations in airflow, domino spacing, surface friction, and sensor timing cause each run to diverge. Sometimes the balloon rushes forward; sometimes it hesitates; sometimes it misses the needles entirely. These deviations become the expressive core of the piece, revealing how unpredictability animates even the most tightly orchestrated mechanical chains.

As an early exploration into physical computation, the machine was prototyped using Arduino to bridge mechanical movement with digital logic, documenting how interdependence and misfires shape system behavior.
